# Web API Geliştirme

Bu repository, Web API geliştirme üzerine aldığım eğitim sürecinde edindiğim teknik bilgi ve pratikleri özetlemek amacıyla oluşturulmuştur.

## Kullanılan Teknolojiler ve Araçlar

- **ASP.NET Core Web API**
- **HTTP / REST**
- **Entity Framework Core**
- **AutoMapper**
- **Swagger (OpenAPI)**
- **API test araçları (request/response odaklı testler)**
- **Katmanlı mimari (Controller, Service, DTO vb.)**

### Web API Temelleri
- HTTP metodlarının doğru kullanım senaryoları
- Status code’ların anlamı ve doğru response üretimi
- RESTful endpoint tasarımı

### API Tasarımı
- Resource bazlı endpoint kurgulama
- Route yönetimi
- DTO kullanımı ile dış dünyaya açılan modelleri ayırma
- Controller’ların sade tutulması

### Veri ve Veritabanı
- Entity modelleme
- API ile veritabanı entegrasyonu
- CRUD operasyonlarının uygulanması
- EF Core ile veri erişimi

### AutoMapper
- Entity ↔ DTO dönüşümlerini yönetme
- Mapping işlemlerini merkezi ve okunabilir hale getirme
- Kod tekrarını azaltma

### Test ve Hata Ayıklama
- API endpoint’lerini manuel olarak test etme
- Farklı senaryolar için istekler oluşturma
- Hata yakalama ve debug süreçlerini yönetme

### Dokümantasyon
- Swagger ile otomatik API dokümantasyonu
- API’yi başka geliştiricilerin anlayabileceği şekilde sunma
- Versiyonlama ihtiyacını ve temel stratejileri kavrama
